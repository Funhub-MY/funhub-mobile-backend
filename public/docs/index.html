Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Login

Login user into the system with email and password and returns user's token used for API authentication

Example request:
curl --request POST \
    "http://localhost/api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"issac81@example.org\",
    \"password\": \"0xT4HqKfcv@iB)Cp0\"
}"

const url = new URL(
    "http://localhost/api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "issac81@example.org",
    "password": "0xT4HqKfcv@iB)Cp0"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, success):


{
 "user": {
    id: 1,
    name: "John Smith"
 },
 "token": "AuthenticationTokenHere"
}

Example response (401, Invalid Login details):


{
    "message": "Invalid login details"
}

Example response (422, Invalid Form Fields):


{"errors": ["email": ["The email field is required."], "password": ["The password field is required."] ]}

Register with Email

Register user with email and password

Example request:
curl --request POST \
    "http://localhost/api/v1/register/email" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"praesentium\",
    \"email\": \"mswaniawski@example.com\",
    \"password\": \"uv(q+m*A.@7,Wb\\/XQ\\\"\"
}"

const url = new URL(
    "http://localhost/api/v1/register/email"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "praesentium",
    "email": "mswaniawski@example.com",
    "password": "uv(q+m*A.@7,Wb\/XQ\""
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, success):


{
 "user": {
    id: 1,
    name: "John Smith"
 },
 "token": "AuthenticationTokenHere"
}

Example response (422, Invalid Form Fields):
