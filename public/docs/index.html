Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Send OTP

Send SMS OTP to phone number

Example request:
curl --request POST \
    "http://localhost/api/v1/sendOtp" \
    --header "Content-Type: application/json" \
    --header "Accept: application/x-www-form-urlencoded" \
    --data "{
    \"country_code\": \"60\",
    \"phone_no\": \"1234567890\"
}"

const url = new URL(
    "http://localhost/api/v1/sendOtp"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/x-www-form-urlencoded",
};

let body = {
    "country_code": "60",
    "phone_no": "1234567890"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, success):


{
    "message": "OTP Sent!"
}

Example response (422, Invalid Form Fields):


{"errors": ["country_code": ["The Country COde field is required."], "phone_no": ["The Phone No field is required."] ]}
Received response:

Login with Password

Example request:

    "http://localhost/api/v1/loginWithPassword" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Accept:: application/x-www-form-urlencoded" \
    --data "{
    \"country_code\": \"60\",
    \"phone_no\": \"1234567890\",
    \"password\": \"abcd1234\"
}"

    "http://localhost/api/v1/loginWithPassword"
);
